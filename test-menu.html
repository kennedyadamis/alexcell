Teste do Menu Mobile

Este é um teste isolado do menu mobile para verificar se está funcionando corretamente.

Como testar:

  1. Redimensione a janela para menos de 768px de largura (ou use F12 → modo responsivo)
  2. O botão ☰ deve aparecer no canto superior direito
  3. Clique no botão ☰
  4. O menu deve expandir com fundo escuro e links brancos
  5. Clique em qualquer link para fechar o menu
